# BCode Lang

**BCode Lang** — минимальный интерпретатор собственного языка программирования на Python.  
Язык поддерживает:
- Переменные (глобальные и локальные)
- Условные конструкции: `if`, `elif`, `else`
- Циклы: `for(arr as item)`
- Функции: `func name(args) { ... }`
- Словари и массивы
- JSON: `json_encode()`, `json_decode()`
- HTTP-запросы: `http_get()`, `http_post()`
- Встроенные функции: `print()`, `input()`, арифметика и логические операторы
